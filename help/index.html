ヘルプ / よくある質問

「XUIDの取得ができませんでした」と表示される

以下の可能性があります：

  • プレイヤー名の誤字。
  • 対象プレイヤーがMicrosoftアカウントに存在しない。
  • BE/Javaの選択を間違えている（特にBEでこのエラーが出ます）。
  • GeyserMC APIが一時的にダウンしている。

「スキンの取得が出来ませんでした」と表示される

以下の原因が考えられます：

  • プレイヤーがスキンを設定していない。
  • 対象プレイヤーのアカウントが存在しない。
  • GeyserMC API または Minecraft API がダウンしている。

「スキンがスティーブやアレックスです」

表示されたスキンが初期スキン（スティーブ／アレックス）の場合、以下を確認してください：

  • プレイヤーがスキンをアップロードしていない。
  • 対象のアカウントが存在しない／使用されていない。
  • API側の問題ではありません（取得は成功している状態です）。

その他のヒント

  • プレイヤー名は大文字・小文字を区別しないが、正確に入力してください。
  • 問題が解決しない場合、時間を置いて再試行してください。
  • BEで問題が出る場合、Javaで試すとヒントになることがあります。
